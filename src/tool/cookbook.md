## 茄汁大虾
- 材料： 虾仁，蒜末（5-6颗）
- 料汁：2勺耗油，1勺生抽，2勺陈醋，4勺番茄酱（没有番茄酱可以用白砂糖替代）
::: info 做法
  1. 虾仁开背煎香（7成熟）
  2. 倒出虾仁，用余油煸香蒜末
  3. 倒入虾仁，倒入料汁，大火收汁，快速翻炒
:::

## 鹿茸菇鸡汤
- 材料：鸡腿肉 *2，鸡翅膀 *2，鹿茸菇，枸杞，红枣（如果有）
- 调料： 沙姜适量，酱油适量
::: info 做法
  1. 鸡淀粉水洗净，鹿茸菇洗净
  2. 放入鹿茸菇，适量水，烧至水开
  3. 放入鸡，15分钟放入枸杞红枣
  4. 放盐，胡椒粉，鸡精（如果有）
:::

## 上汤娃娃菜
- 材料：鹿茸菇鸡汤适量，娃娃菜，枸杞
- ::: info 做法
  1. 娃娃菜洗净
  2. 放入鹿茸菇鸡汤，娃娃菜，大火烧至水开
  3. 放枸杞，中小火焖煮3-5分钟
  4. 放盐，胡椒粉
:::
